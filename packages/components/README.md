## Install

Using `npm` to install:

```bash
# install latest version for Vue 3 project
npm i vafar
```

Using `yarn`, `pnpm`:

```bash
# with yarn
yarn add vafar

# with pnpm
pnpm add vafar
```

## Quickstart

```ts
import { createApp } from 'vue';

import  Vafar from 'vafar';

import  'vafar/ui/es/style.css'

const app = createApp();

app.use(Vafar);
```
